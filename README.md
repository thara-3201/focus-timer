# ✿ About

Focus Time is a soft and calming desktop focus timer designed to make productivity feel cozy instead of overwhelming.

Built with:
- gentle colors
- minimal distractions
- scrapbook-inspired visuals
- simple controls

Perfect for:
- study sessions
- deep work
- journaling time
- reading
- mindful productivity

# ✿ Features

🌿 Minimal & calming UI  
🌸 Cute scrapbook-inspired aesthetic  
⏳ Focus timer with pause/reset

# ✿ Installation

## macOS

1. Download the `.dmg`
2. Open it
3. Drag **Focus Time** into Applications

---

## Windows

1. Download the `.exe`
2. Open installer
3. Click “Run Anyway” if Windows shows a warning

---

## Linux

```bash
chmod +x FocusTime.AppImage
./FocusTime.AppImage
```
